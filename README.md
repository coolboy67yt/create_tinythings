# Create: Tiny Things
Addon for Create mod, with Kinetic Batteries, mechanical pets, and new tools.

## Features
- Kinetic Batteries<br>
Store rotational energy in a portable battery!
- Mechanical Pets<br>
A lil guy who follows you around and carries items for you :]
